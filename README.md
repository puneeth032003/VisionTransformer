## Overview
This project implements a **Vision Transformer (ViT)** for image classification on CIFAR-10.
It demonstrates how transformers, originally for NLP, can be adapted to vision tasks.

## Features
- Modular, well-structured code (src folder)
- Reproducible training and evaluation
- One-click Colab execution
- Unit tests for critical components

## Dataset
- **CIFAR-10**: 60,000 32x32 color images in 10 classes
  - Training images: 50,000
  - Test images: 10,000
